## Syntastic
> Add [Syntastic](https://github.com/scrooloose/syntastic) to your Plugins. <br>
> This example uses [Plug](https://github.com/junegunn/vim-plug)

```vim
Plug 'scrooloose/syntastic'
Plug 'pmsorhaindo/syntastic-local-eslint.vim'
```

> Install the plugin

```vim
:PlugInstall
```
